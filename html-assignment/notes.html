List of HTML tags

<html> </html>:
Declares the Web page to be written in HTML.
<title> </title>:
Defines the title.
<b> </b>:
Set ... in boldface.
<div> </div>:
Divides the content into logical groups which will show the content on a separate line.
<span> </span>:
Shows the content inline with the rest of the content.
